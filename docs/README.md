# Relevant Documents

This directory contains the following important documents related to the ICARUS project:

- `article.pdf`: This document is a draft of a research article intended for publication. It summarizes the key aspects of the ICARUS project, including its objectives, methodologies, and findings.
- `dissertation.pdf`: This is the main dissertation document that led to the development of the ICARUS system. It provides a comprehensive overview of the ICARUS project, including its objectives, methodologies, results, and conclusions.
