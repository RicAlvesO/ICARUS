# ICARUS' Agent

This folder contains the source code and related files for the ICARUS Agent component. This Agent is responsible for collecting and sending data from the monitored systems to the ICARUS Server for analysis, correlation, and threat detection.

## Folder Structure

- `src/`: Contains the source code for the ICARUS Agent, including query management, data collection, and communication modules.
- `data/`: Contains configuration files, query definitions, and other data required by the Agent.
- `mon-agent`: The main executable for the ICARUS Agent. 
- `requirements.txt`: A list of Python dependencies required to run the ICARUS Agent.
